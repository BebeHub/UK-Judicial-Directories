# Description
This dataset comprises contemporaneous Website capture as PDF of the UK Judges directory, at specific intervals between June 2025 and April 2026 and a more comprehensive list of Deputy Judges made available to the public in November 2020 as result of a Freedom of Information (FOI) request. Each file provides the source/website and date in its header.
 
In socio-legal research, the primary evidentiary value of these documents lies in their original form and temporal context. The UK Judiciary website is a dynamic resource where historical lists are frequently overwritten by updates. 

Successive versions of the list document changes in judicial composition over time, including appointments, departures, and circuit reassignments. Where the official record is silently overwritten without versioning or archiving, these contemporaneous captures constitute the only retrievable evidence of the directory's state at each point in time.
 
This archive therefore supports longitudinal analysis of judicial deployment patterns and serves as a primary source for accountability and transparency research.
